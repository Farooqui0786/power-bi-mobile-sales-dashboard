# 📊 Mobile Sales Dashboard – Power BI Project

## Overview

This project is a Mobile Sales Dashboard created in Power BI to analyze and understand mobile phone sales data in a simple and interactive way. The dashboard helps in tracking sales performance, customer behavior, payment methods, product demand, and city-wise sales distribution.

The main purpose of this dashboard is to turn raw sales data into useful insights that can help businesses make better decisions. Instead of checking large Excel sheets manually, this dashboard provides all important information in one place with charts, graphs, maps, and filters.

---

# 🚀 Features of the Dashboard

### ✅ KPI Cards

The dashboard includes important KPI cards that quickly show:

* Total Sales
* Total Quantity Sold
* Total Transactions
* Average Sales

These cards help users understand the overall business performance at a glance.

---

### 🌍 City-wise Sales Map

A map visualization is used to show sales across different cities.
This helps in identifying:

* High performing cities
* Low performing areas
* Regional sales trends

---

### 📈 Monthly Sales Trend

A line chart is used to display sales quantity trends over time.
This helps users understand:

* Sales growth
* Monthly performance
* Seasonal trends

---

### 📊 Mobile Model Sales Comparison

A clustered column chart compares total sales for different mobile models.
This visualization helps in:

* Identifying best-selling mobile models
* Comparing product performance
* Understanding customer preferences

---

### 💳 Payment Method Analysis

A pie chart is used to show different payment methods used by customers.
This helps businesses understand:

* Most preferred payment methods
* Customer purchasing behavior
* Digital payment trends

---

### ⭐ Customer Ratings Analysis

A funnel chart is added to analyze customer ratings.
This helps in understanding:

* Customer satisfaction
* Product quality feedback
* Service performance

---

### 📅 Day-wise Sales Analysis

A stacked area chart is used to analyze sales performance by day.
This helps in:

* Identifying busy sales days
* Tracking weekly trends
* Improving sales planning

---

### 📋 Detailed Sales Table

A table visualization is included for detailed data analysis.
The table contains:

* Brand
* Total Sales
* Quantity Sold
* Transactions

This allows users to check detailed records easily.

---

### 🎛 Interactive Filters / Slicers

The dashboard includes multiple slicers for better analysis.
Users can filter data based on:

* Brand
* Mobile Model
* Payment Method
* Other sales-related fields

These filters make the dashboard interactive and user-friendly.
